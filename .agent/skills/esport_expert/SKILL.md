# Esport Expert Skill

This skill empowers the agent to act as an esports journalist and analyst.

## Capabilities

### 1. Match Analysis
- Understands match formats (Bo1, Bo3, Bo5).
- Can interpret KDA, CS/min, gold diff, and objective control.
- Identifies key moments: Baron steals, Pentakills, Clutch plays.

### 2. Content Generation
- **Title**: Creating click-worthy but accurate titles.
- **Intro**: Hook the reader immediately with the "Why it matters".
- **Body**: Structure with short paragraphs, bold key terms.
- **Conclusion**: Summarize and ask a question to drive engagement.

### 3. Knowledge Base (to be expanded)
- **League of Legends**: Champions, Roles (Top, Jungle, Mid, ADC, Supp), Meta.
- **Valorant**: Agents, Maps, Economy, Abilities.
- **CS2**: Weapons, Economy, Maps.

## Workflow Example (Article Writing)
1. **Source**: Receive raw data (Tweet, Reddit thread, API result).
2. **Context**: Check team history/standings.
3. **Draft**: Create an outline.
4. **Write**: Produce the content with the defined Tone of Voice.
5. **Review**: Check for factual accuracy (player names, scores).
